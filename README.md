新思路：

1.直接关闭进程（希沃自己有保护，所以不太行[但是可以利用驱动进ring0，值得尝试]，还有就是dll注入）

2.利用fidder进行抓包（解锁时的数据包和端口）再用winsocket发送数据包（逝逝就逝逝）
